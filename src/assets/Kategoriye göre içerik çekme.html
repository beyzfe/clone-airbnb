1. URL Parametresi ile Kategori Verisini Alma

Öncelikle, category/:categoryName gibi bir route tanımladığını varsayalım. Kategoriye tıklandığında, bu kategorinin adını URL'den alabilmek için useParams hook'unu kullanabilirsin.

tsx
import { useParams } from "react-router-dom";

const CategoryPage = () => {
  const { categoryName } = useParams(); // URL'deki categoryName parametresini al

  // Burada categoryName'e göre veri çekebilirsin
  console.log(categoryName); // Örneğin: "technology"

  return (
    <div>
      <h1>{categoryName} Kategorisi</h1>
      {/* Kategoriye ait içerik burada render edilecek */}
    </div>
  );
};

2. Kategoriye Ait Verileri API veya Yerel Verilerle Çekmek

Eğer verileri bir API'den çekmek istiyorsan, useEffect ve fetch kullanabilirsin. Örneğin:

tsx
import { useState, useEffect } from "react";
import { useParams } from "react-router-dom";

const CategoryPage = () => {
  const { categoryName } = useParams();
  const [categoryData, setCategoryData] = useState<any>(null);

  useEffect(() => {
    // API çağrısını buraya ekleyebilirsin
    const fetchCategoryData = async () => {
      try {
        const response = await fetch(`/api/categories/${categoryName}`);
        const data = await response.json();
        setCategoryData(data);
      } catch (error) {
        console.error("Hata:", error);
      }
    };

    fetchCategoryData();
  }, [categoryName]); // categoryName değiştikçe yeniden veri çek

  if (!categoryData) {
    return <div>Loading...</div>;
  }

  return (
    <div>
      <h1>{categoryName} Kategorisi</h1>
      <ul>
        {categoryData.items.map((item: any) => (
          <li key={item.id}>{item.name}</li> // Kategorideki öğeleri listeler
        ))}
      </ul>
    </div>
  );
};

tsx

const App = () => {

    <Router>
      <Routes>
        <Route path="/category/:categoryName" element={<CategoryPage />} />
      </Routes>
    </Router>
  );
};

Özet:

  1. useParams ile URL'deki kategori adını al.

  2. useEffect ile kategoriye ait verileri bir API'den ya da yerel veriden çek.

  3. Verileri render et ve kategoriye özel içerik göster.

Evet, şimdi tıklanan kategoriye ait API'den veri çekip, bu veriyi ekranda göstermek için birkaç adım izleyebiliriz. İşte bunun için takip etmen gereken adımlar:

1. API'den Veri Çekme

Öncelikle, tıklanan kategoriye ait verileri API'den çekmek için useEffect ve fetch (ya da axios gibi başka bir kütüphane) kullanacağız.

2. useEffect ile Veri Çekme

React useEffect hook'u, bileşenin render edilmesinden sonra veri çekme işlemi için kullanılır. Bu sayede, kategori adı değiştikçe (URL değiştikçe) API'yi çağırabiliriz.

3. API Çağrısı ve Veriyi Ekranda Gösterme

Örnek olarak, kategorinin ID'sine göre API çağrısı yapalım ve dönen veriyi ekranda gösterelim.

tsx
import { useParams } from "react-router-dom";
import { useState, useEffect } from "react";

const CategoryPage = () => {
  const { categoryName } = useParams(); // URL'deki 'categoryName' parametresini al
  const [categoryData, setCategoryData] = useState<any>(null); // API'den gelen veriyi tutmak için state
  const [loading, setLoading] = useState<boolean>(true); // Veri yükleniyor mu durumunu tutmak için state
  const [error, setError] = useState<string | null>(null); // Hata durumu için state

  useEffect(() => {
    const fetchCategoryData = async () => {
      setLoading(true); // Yükleme işlemi başladığında true yapıyoruz
      setError(null); // Hata durumu sıfırlanır

      try {
        // API'ye çağrı yapıyoruz. Burada URL dinamik olmalı, örneğin:
        const response = await fetch(`/api/categories/${categoryName}`);

        if (!response.ok) {
          throw new Error("Veri yüklenirken bir hata oluştu");
        }

        const data = await response.json(); // API'den gelen veriyi JSON olarak parse et
        setCategoryData(data); // Veriyi state'e kaydet
      } catch (err) {
        setError("Bir hata oluştu: " + err); // Hata varsa göster
      } finally {
        setLoading(false); // Yükleme işlemi tamamlandı
      }
    };

    fetchCategoryData(); // Veriyi çekmeye başla
  }, [categoryName]); // categoryName değiştiğinde yeniden çalışacak

  if (loading) {
    return <div>Loading...</div>; // Yükleniyor mesajı göster
  }

  if (error) {
    return <div>{error}</div>; // Hata mesajı göster
  }

  // API'den gelen veriyi ekranda göster
  return (
    <div>
      <h1>Tıklanan Kategori: {categoryName}</h1>
      {categoryData ? (
        <div>
          <h2>Veriler:</h2>
          <ul>
            {categoryData.items.map((item: any) => (
              <li key={item.id}>{item.name}</li> // Kategorideki öğeleri listele
            ))}
          </ul>
        </div>
      ) : (
        <div>Bu kategoriye ait veri bulunamadı.</div>
      )}
    </div>
  );
};

export default CategoryPage;

Adımların Açıklaması:

  1. useParams: URL'deki categoryName parametresini alıyoruz. Bu, hangi kategoriye tıklandığını belirleyecek.

  2. useEffect: categoryName değiştikçe API'yi çağıracak şekilde ayarlandı. Bu sayede her kategoriye tıklandığında veriyi dinamik olarak çekecek.

  3. API Çağrısı: fetch fonksiyonu ile, categoryName'e göre ilgili kategoriye ait veriyi çeken bir API çağrısı yapıyoruz. Bu örnekte API URL'si /api/categories/${categoryName} şeklinde. Gerçek API'yi kullanırken bu URL'yi değiştirebilirsin.

  4. Yükleme Durumu: API verisi çekilirken loading state'i ile yükleniyor mesajı gösteriyoruz.

  5. Hata Durumu: Eğer API çağrısında bir hata olursa, hata mesajı gösterilecek.

  6. Veri Gösterimi: API'den gelen veriyi ekranda liste olarak gösteriyoruz. Burada categoryData.items kullanarak gelen veriyi listeliyoruz (veri formatı API'ye bağlı olarak değişebilir).

4. API Endpoint

Tabii ki, burada kullandığım /api/categories/${categoryName} endpoint'i sadece bir örnek. Gerçek API'nizin endpoint'ini ve veri yapısını göz önünde bulundurarak gerekli düzenlemeleri yapmalısın.

API'den Verileri Çekme ve Filtreleme

Şimdi, tıklanan kategoriye ait ürünleri filtreleyip göstereceğiz. Önce CategoryPage bileşenini sadeleştirelim:

TypeScript ile CategoryPage:

tsx
import { useParams } from "react-router-dom";
import { useState, useEffect } from "react";

// Ürün verisi için tip
interface Product {
  id: number;
  title: string;
  price: number;
  description: string;
  category: string;
  image: string;
  rating: {
    rate: number;
    count: number;
  };
}

// Kategoriler için tip
type Category = string;

const CategoryPage = () => {
  const { categoryName } = useParams<{ categoryName: string }>(); // URL'deki kategori ismini al
  const [products, setProducts] = useState<Product[]>([]); // Ürünleri tutacak state
  const [loading, setLoading] = useState<boolean>(true); // Veri yükleniyor durumu

  useEffect(() => {
    const fetchProducts = async () => {
      setLoading(true);
      try {
        // Örnek API URL'si
        const response = await fetch("https://fakestoreapi.com/products");
        const data: Product[] = await response.json();
        
        // Kategoriye göre filtreleme
        const filteredProducts = data.filter(product => product.category === categoryName);
        
        setProducts(filteredProducts); // Filtrelenmiş ürünleri state'e kaydet
      } catch (error) {
        console.error("Veri çekme hatası:", error);
      } finally {
        setLoading(false);
      }
    };

    fetchProducts();
  }, [categoryName]); // Kategori değiştikçe veri çek

  if (loading) {
    return <div>Loading...</div>; // Veri yükleniyorsa "Loading..." mesajı göster
  }

  return (
    <div>
      <h1>{categoryName} Kategorisi</h1>
      {products.length === 0 ? (
        <p>Bu kategoride ürün bulunmamaktadır.</p>
      ) : (
        <ul>
          {products.map((product) => (
            <li key={product.id} className="mb-4 p-4 border">
              <h2>{product.title}</h2>
              <img src={product.image} alt={product.title} className="w-32" />
              <p>{product.description}</p>
              <p><strong>Price:</strong> ${product.price}</p>
              <p><strong>Rating:</strong> {product.rating.rate} ({product.rating.count} reviews)</p>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
};

export default CategoryPage;

Açıklamalar:

  1. Product Tipi: Ürün verisini daha güvenli şekilde yönetmek için Product tipini oluşturduk.

  2. Veri Çekme (useEffect): useEffect hook'u, URL'deki kategori adı değiştikçe API'den ürün verilerini çekiyor. fetch ile API'den gelen veriyi aldıktan sonra, tıklanan kategoriye ait ürünleri filtreliyoruz.

  3. categoryName: URL'den alınan categoryName parametresine göre, Product[] dizisini filtreliyoruz.

  4. products State'i: Filtrelenmiş ürünleri products state'inde tutuyoruz.

  5. Yükleniyor Durumu: Veri yüklenene kadar "Loading..." mesajı gösteriyoruz.

  6. Veri Gösterimi: Eğer kategoriye ait ürünler varsa, her bir ürünü liste olarak ekranda gösteriyoruz.
